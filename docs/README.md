# 문서 개요

이 디렉터리는 CNC 시뮬레이터의 사용자/개발 문서를 위한 폴더입니다.

## 이번 버전의 핵심 포인트

- **가공 흔적 맵**: 절삭 스윕이 지난 영역을 footprint 형태로 누적 표시
- **AE/AP 기반 해석**: 스톡 기반 맞물림 추정으로 스핀들 부하와 채터 위험도 계산
- **X/Y/Z 축 진동 정보**: 축강성 기반 예상 진동(um) 표시
- **소재 원점/크기 설정**: origin + size + origin_mode 방식 지원

## 참고 문서

- `README.md`: 프로젝트 개요, 실행 방법, 설정 항목
- `ARCHITECTURE.md`: 주요 모듈 구조와 데이터 흐름

## 사용 시 주의

- 소재 모델은 Z-map 방식이므로 격자 해상도에 따른 근사 오차가 있습니다.
- 채터 위험도는 SLD 완전 해석이 아니라 공학적 근사 평가입니다.
- 실제 생산 적용 전에는 실측 데이터와 장비 조건으로 반드시 재검토해야 합니다.
